[TR] Uygulama ayağa kaldırıldığında Admin rolünde kullanıcı adı 'admin', şifresi 'pass' olan bir hesap otomatik olarak oluşturulur. Bu hesaba giriş yaparak öğrenci ve öğretmen rolündeki hesaplar oluşturabilirsiniz.

[EN] When you run the backend application, a new admin account with 'admin' username and 'pass' password is created. You can login that account and create student/instructor accounts.
